classtemplater

Overview of codegeneration process
Opening classtemplater for Person class

Download

Update site: http://classtemplater.inepex.com

Usage

1. Create a template file in your workspace with .vm extension. For example: 

hello world
#foreach( $attr in $attrs )
${attr.name} - ${attr.type} 
#end
It is a Velocity template. The attr variable contains the attributes. The ${attr.name} variable holds the name of the attribute while ${attr.type} is the type of the attribute.

2. Select a java class in Package Explorer, hit right mouse button and choose "classtemplater/Generate..."

3. In the classtemplater window select your template

4. The generated code shows up in the bottom window and gets to the clipboard.

Features

  • save output to file automatically
  • generate code based more than one classes with one step
  • generate code based more than one templates with one step
  • generate code based multiple classes and multiple templates with one step
  • handwritten parts in generated code
  • annotation support for base class
