classtemplater

Overview of codegeneration process
Opening classtemplater for Person class

Letöltés

Update site-nak ezt kell megadni: http://sotiwin.hu/classtemplater/

Használat

1. Hozz létre a workspace-edben egy template fájlt. Pl: 

hello world
#foreach( $attr in $attrs )
${attr.name} - ${attr.type} 
#end
A fájl kiterjesztése vm legyen.
Ez velocity template nyelven íródott. Az attrs változó-val érheted el az attribútumokat. A ${attr.name} változó az attribútum neve, az ${attr.type} az attribútum típusa.

2. A Package Explorer-ben válassz ki egy .java file-t nyomj jobbgombot és a menüből válaszd a classtemplater/Generate...-et

3. A felugró ablakban kattints a template-re

4. A generált kód azonnal megjelenik az ablakban és a vágólapra másolódik.

További funkciók

  • automatikus fájlba mentés
  • generálás egyszerre több osztályhoz
  • generálás egyszerre több template-el
  • generálás egyszerre több osztályhoz, több template-el
  • a generált kód kézi módosításának támogatása
  • annotációk támogatása
